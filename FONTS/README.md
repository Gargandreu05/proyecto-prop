COM PROVAR EL PROJECTE
A continuació es posaran les comandes per a gradle, per a provar, compilar i construir el projecte.

./gradlew test: executarà els tests unitaris.

./gradlew javadoc: generarà la documentació javadoc en la carpeta DOCS.

./gradlew build: compilarà i executarà els testos del projecte.

./gradlew clean: netejarà els fitxers de compilació i els artefactes creats.

Per executar el codi:

./gradlew runEnquestaDriver: executarà el Enquesta Driver.

./gradlew runUsuariDriver: executarà el Usuari Driver.

Per a fer tests amb Windows, canviar el fitxer executat "gradlew" per "gradlew.bat"
