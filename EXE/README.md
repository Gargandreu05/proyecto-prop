- A WINDOWS

Només cal fer doble click (obrir) el fitxer .bat del driver que es desitja executar i s'executarà.

- A LINUX

Cal obrir una terminal en aquest fitxer, i es necessita guarantir permisos d'execució al fitxers .sh en cas que no els tenguin:

chmod +x *.sh

Això només cal fer-ho una vegada.

Una vegada tenen permisos d'execució només cal executar els respectius fitxers segons el driver que es vulgui testejar.

./runEnquestaDriver.sh
